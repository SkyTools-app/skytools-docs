## GET /v1/auctions/lowest-bins

Returns the lowest BIN (Buy It Now) price for every item currently listed on the auction house. Scans up to 20 pages of auctions to find the cheapest listing for each item.

**Cache:** 5 minutes

### Query parameters

None.

### Examples

<CodeGroup>
```bash cURL
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/auctions/lowest-bins"
```

```javascript JavaScript
const response = await fetch("https://api.skytools.app/v1/auctions/lowest-bins", {
  headers: { "X-API-Key": "sk_your_key_here" },
});
const { data } = await response.json();

// Look up a specific item
const hyperionPrice = data.lowestBins["Hyperion"];
console.log(`Cheapest Hyperion: ${hyperionPrice.toLocaleString()} coins`);
```

```python Python
import requests

response = requests.get(
    "https://api.skytools.app/v1/auctions/lowest-bins",
    headers={"X-API-Key": "sk_your_key_here"},
)
lowest_bins = response.json()["data"]["lowestBins"]

hyperion_price = lowest_bins.get("Hyperion", "Not listed")
print(f"Cheapest Hyperion: {hyperion_price:,} coins")
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "lowestBins": {
      "Hyperion": 900000000,
      "Aspect of the Dragons": 15000000,
      "Livid Dagger": 30000000,
      "Necron's Handle": 450000000
    }
  },
  "meta": { "cached": true, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `lowestBins` | object | Map of item display name to lowest BIN price in coins |

Keys are **item display names** (not internal IDs). Values are the lowest BIN price in coins as an integer.
