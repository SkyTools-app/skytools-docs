## GET /v1/player/:username

Returns general Hypixel player stats including network level, karma, rank, achievement points, online status, and login timestamps. All fields are computed/derived from raw Hypixel data.

**Cache:** 5 minutes

### Path parameters

| Parameter | Type | Description |
|---|---|---|
| `username` | string | Minecraft username (1-16 chars, alphanumeric and underscores only) |

### Examples

<CodeGroup>
```bash cURL
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/player/Technoblade"
```

```javascript JavaScript
const response = await fetch("https://api.skytools.app/v1/player/Technoblade", {
  headers: { "X-API-Key": "sk_your_key_here" },
});
const { data } = await response.json();
console.log(`${data.username} [${data.rankFormatted}] Level ${data.level}`);
console.log(`Online: ${data.online}`);
```

```python Python
import requests

response = requests.get(
    "https://api.skytools.app/v1/player/Technoblade",
    headers={"X-API-Key": "sk_your_key_here"},
)
data = response.json()["data"]
print(f"{data['username']} [{data['rankFormatted']}] Level {data['level']}")
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "uuid": "b876ec32e396476ba1158438d83c67d4",
    "username": "Technoblade",
    "rank": "MVP_PLUS",
    "rankFormatted": "MVP+",
    "level": 256.72,
    "karma": 12500000,
    "achievementPoints": 8450,
    "questsCompleted": 1200,
    "challengesCompleted": 5000,
    "giftsGiven": 15,
    "ranksGiven": 3,
    "firstLogin": 1420070400000,
    "lastLogin": 1710600000000,
    "lastLogout": 1710590000000,
    "online": true,
    "totalExp": 84250000
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `uuid` | string | Player's Minecraft UUID (dashless) |
| `username` | string | Player's current username |
| `rank` | string | Resolved rank code (`MVP_PLUS`, `SUPERSTAR`, `NONE`, etc.) |
| `rankFormatted` | string | Human-readable rank name (`MVP+`, `MVP++`, `VIP`, `Default`, etc.) |
| `level` | number | Precise Hypixel network level (computed from XP) |
| `karma` | integer | Karma points |
| `achievementPoints` | integer | Total achievement points |
| `questsCompleted` | integer | Total quests completed (computed from achievements) |
| `challengesCompleted` | integer | Total challenges completed (computed from achievements) |
| `giftsGiven` | integer | Number of gifts given |
| `ranksGiven` | integer | Number of ranks gifted to other players |
| `firstLogin` | integer | First login timestamp (Unix ms) |
| `lastLogin` | integer | Last login timestamp (Unix ms) |
| `lastLogout` | integer | Last logout timestamp (Unix ms) |
| `online` | boolean | Whether the player is currently online (computed: `lastLogin > lastLogout`) |
| `totalExp` | integer | Raw network experience points |

<Note>The `rank` field uses the Hypixel rank priority chain: `rank` > `monthlyPackageRank` > `newPackageRank` > `packageRank`. The `rankFormatted` field converts this to a human-readable name (e.g., `SUPERSTAR` becomes `MVP++`).</Note>

## GET /v1/player/:username/bedwars

Returns BedWars stats with computed ratios, win rates, prestige tier, and per-mode breakdowns.

**Cache:** 5 minutes

### Path parameters

| Parameter | Type | Description |
|---|---|---|
| `username` | string | Minecraft username |

### Example

```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/player/Technoblade/bedwars"
```

### Response

```json
{
  "success": true,
  "data": {
    "uuid": "b876ec32e396476ba1158438d83c67d4",
    "username": "Technoblade",
    "level": 245,
    "prestige": "Stone",
    "coins": 850000,
    "overall": {
      "wins": 1200, "losses": 800,
      "winRate": 60.0, "wlr": 1.5,
      "kills": 5000, "deaths": 3000, "kdr": 1.67,
      "finalKills": 2500, "finalDeaths": 1000, "fkdr": 2.5,
      "bedsBroken": 1800, "bedsLost": 900, "bblr": 2.0,
      "winstreak": 5, "gamesPlayed": 2000
    },
    "modes": {
      "solo": { "wins": 300, "losses": 200, "winRate": 60.0, "wlr": 1.5, "kills": 1200, "deaths": 800, "kdr": 1.5, "finalKills": 600, "finalDeaths": 250, "fkdr": 2.4, "bedsBroken": 450, "bedsLost": 220, "bblr": 2.05, "winstreak": 2, "gamesPlayed": 500 },
      "doubles": { ... },
      "threes": { ... },
      "fours": { ... }
    }
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `level` | integer | BedWars star level (from achievements) |
| `prestige` | string | Computed prestige tier: `None`, `Stone`, `Iron`, `Gold`, `Diamond`, `Emerald` |
| `coins` | integer | BedWars coins |
| `*.wins` | integer | Games won |
| `*.losses` | integer | Games lost |
| `*.winRate` | number \| null | Win percentage (computed: `wins / gamesPlayed * 100`) |
| `*.wlr` | number \| null | Win/Loss ratio (computed) |
| `*.kills` | integer | Kills |
| `*.deaths` | integer | Deaths |
| `*.kdr` | number \| null | Kill/Death ratio (computed) |
| `*.finalKills` | integer | Final kills |
| `*.finalDeaths` | integer | Final deaths |
| `*.fkdr` | number \| null | Final Kill/Death ratio (computed) |
| `*.bedsBroken` | integer | Beds destroyed |
| `*.bedsLost` | integer | Beds lost |
| `*.bblr` | number \| null | Beds Broken/Lost ratio (computed) |
| `*.winstreak` | integer | Current winstreak |
| `*.gamesPlayed` | integer | Total games played (computed: `wins + losses`) |

<Note>Ratios return `null` when the divisor is 0 (e.g., 0 deaths), or `Infinity` when the numerator is positive but the divisor is 0 (e.g., 5 wins, 0 losses). Modes: `solo`, `doubles`, `threes`, `fours`.</Note>

### Errors (all player endpoints)

| Code | HTTP Status | When |
|---|---|---|
| `BAD_REQUEST` | 400 | Invalid username format (must be 1-16 chars, `[a-zA-Z0-9_]`) |
| `NOT_FOUND` | 404 | Player not found on Mojang/Hypixel |
| `UPSTREAM_ERROR` | 502 | Hypixel or Mojang API returned an error |
| `TIMEOUT` | 503 | Upstream timed out and no cached data available |
