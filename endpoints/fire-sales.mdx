## GET /v1/fire-sales

Returns currently active fire sales from the Hypixel API.

**Cache:** 1 minute

### Query parameters

None.

### Examples

<CodeGroup>
```bash cURL
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/fire-sales"
```

```javascript JavaScript
const response = await fetch("https://api.skytools.app/v1/fire-sales", {
  headers: { "X-API-Key": "sk_your_key_here" },
});
const { data } = await response.json();

if (data.sales.length === 0) {
  console.log("No active fire sales right now");
} else {
  data.sales.forEach((sale) => {
    console.log(`${sale.item_id}: ${sale.amount} available at ${sale.price} gems`);
  });
}
```

```python Python
import requests

response = requests.get(
    "https://api.skytools.app/v1/fire-sales",
    headers={"X-API-Key": "sk_your_key_here"},
)
sales = response.json()["data"]["sales"]

for sale in sales:
    print(f"{sale['item_id']}: {sale['amount']} at {sale['price']} gems")
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "sales": [
      {
        "item_id": "PET_SKIN_DRAGON_NEON_BLUE",
        "start": 1706000000000,
        "end": 1706100000000,
        "amount": 200,
        "price": 800
      }
    ]
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `sales[].item_id` | string | Internal SkyBlock item ID |
| `sales[].start` | number | Sale start timestamp (Unix ms) |
| `sales[].end` | number | Sale end timestamp (Unix ms) |
| `sales[].amount` | integer | Total quantity available |
| `sales[].price` | integer | Price in SkyBlock gems |

## GET /v1/fire-sales/history

Returns past fire sales from the database.

**Cache:** 5 minutes

### Query parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `limit` | integer | `50` | Number of results (1–200) |

### Examples

<CodeGroup>
```bash cURL
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/fire-sales/history?limit=20"
```

```javascript JavaScript
const response = await fetch(
  "https://api.skytools.app/v1/fire-sales/history?limit=20",
  { headers: { "X-API-Key": "sk_your_key_here" } }
);
const { data } = await response.json();
data.history.forEach((sale) => {
  console.log(`${sale.item_id}: ${sale.amount} at ${sale.price} gems (${sale.recorded_at})`);
});
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "history": [
      {
        "item_id": "PET_SKIN_DRAGON_NEON_BLUE",
        "amount": 200,
        "price": 800,
        "recorded_at": "2026-01-15T12:00:00Z"
      }
    ]
  },
  "meta": { "cached": true, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `history[].item_id` | string | Internal SkyBlock item ID |
| `history[].amount` | integer | Total quantity that was available |
| `history[].price` | integer | Price in SkyBlock gems |
| `history[].recorded_at` | string | ISO 8601 timestamp when the sale was recorded |
