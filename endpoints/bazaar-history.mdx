## GET /v1/bazaar/history

Returns historical price data for a bazaar item. Short ranges (1h, 24h) return raw 5-minute snapshots. Longer ranges (7d, 30d, 90d) return hourly aggregates.

**Cache:** 5 minutes

<Warning>
If the upstream data source times out, the API may return stale cached data (up to 30 minutes old). Check `meta.stale` in the response to detect this.
</Warning>

### Query parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `item_id` | string | — | **Required.** Bazaar item ID (e.g. `DIAMOND`) |
| `range` | string | `24h` | Time range: `1h`, `24h`, `7d`, `30d`, `90d` |

### Data resolution

| Range | Source | Max Data Points |
|---|---|---|
| `1h` | 5-minute snapshots | 100 |
| `24h` | 5-minute snapshots | 288 |
| `7d` | Hourly aggregates | 500 |
| `30d` | Hourly aggregates | 720 |
| `90d` | Hourly aggregates | 1,000 |

### Examples

<CodeGroup>
```bash cURL
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar/history?item_id=DIAMOND&range=24h"
```

```javascript JavaScript
const response = await fetch(
  "https://api.skytools.app/v1/bazaar/history?item_id=DIAMOND&range=24h",
  { headers: { "X-API-Key": "sk_your_key_here" } }
);
const { data } = await response.json();
console.log(`${data.dataPoints} data points`);

// Plot the price history
data.history.forEach((point) => {
  console.log(`${point.recorded_at}: sell=${point.sell_price} buy=${point.buy_price}`);
});
```

```python Python
import requests

response = requests.get(
    "https://api.skytools.app/v1/bazaar/history",
    params={"item_id": "DIAMOND", "range": "24h"},
    headers={"X-API-Key": "sk_your_key_here"},
)
history = response.json()["data"]["history"]

for point in history:
    print(f"{point['recorded_at']}: sell={point['sell_price']}")
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "item_id": "DIAMOND",
    "range": "24h",
    "dataPoints": 288,
    "history": [
      {
        "sell_price": 7.5,
        "buy_price": 8.1,
        "sell_volume": 1234567,
        "buy_volume": 987654,
        "spread": 0.6,
        "spread_percent": 8.0,
        "recorded_at": "2026-03-05T12:00:00Z"
      }
    ]
  },
  "meta": { "cached": true, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `sell_price` | number | Instant-sell price at this point in time |
| `buy_price` | number | Instant-buy price at this point in time |
| `sell_volume` / `buy_volume` | number | Order volume at this snapshot |
| `spread` | number | `buy_price - sell_price` |
| `spread_percent` | number | Spread as a percentage of sell price |
| `recorded_at` | string | ISO 8601 timestamp |

### Errors

| Code | HTTP Status | When |
|---|---|---|
| `BAD_REQUEST` | 400 | Missing `item_id` parameter or invalid `range` value |
| `TIMEOUT` | 503 | Upstream data source timed out and no cached data available |
