## GET /v1/recipes

Returns crafting recipes for SkyBlock items, sourced from the NEU (NotEnoughUpdates) repository.

**Cache:** 24 hours

### Query Parameters

None.

### Examples

<Tabs>
  <TabItem label="cURL">
```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/recipes"
```
  </TabItem>
  <TabItem label="JavaScript">
```javascript
const response = await fetch("https://api.skytools.app/v1/recipes", {
  headers: { "X-API-Key": "sk_your_key_here" },
});
const { data } = await response.json();
console.log(`${Object.keys(data).length} recipes loaded`);
```
  </TabItem>
  <TabItem label="Python">
```python
import requests

response = requests.get(
    "https://api.skytools.app/v1/recipes",
    headers={"X-API-Key": "sk_your_key_here"},
)
recipes = response.json()["data"]
print(f"{len(recipes)} recipes loaded")
```
  </TabItem>
</Tabs>

### Response

```json
{
  "success": true,
  "data": { ... },
  "meta": { "cached": true, "api_version": "v1" }
}
```

The response contains crafting recipe data for SkyBlock items, including ingredient lists and crafting grid positions.
