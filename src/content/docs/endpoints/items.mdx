## GET /v1/items

Returns the complete catalog of Hypixel SkyBlock items.

**Cache:** 1 hour

### Query Parameters

None.

### Examples

<Tabs>
  <TabItem label="cURL">
```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/items"
```
  </TabItem>
  <TabItem label="JavaScript">
```javascript
const response = await fetch("https://api.skytools.app/v1/items", {
  headers: { "X-API-Key": "sk_your_key_here" },
});
const { data } = await response.json();
console.log(`${data.items.length} items`);

// Find a specific item
const hyperion = data.items.find((item) => item.id === "HYPERION");
```
  </TabItem>
  <TabItem label="Python">
```python
import requests

response = requests.get(
    "https://api.skytools.app/v1/items",
    headers={"X-API-Key": "sk_your_key_here"},
)
items = response.json()["data"]["items"]
print(f"{len(items)} items loaded")
```
  </TabItem>
</Tabs>

### Response

```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "DIAMOND_SWORD",
        "name": "Diamond Sword",
        "material": "DIAMOND_SWORD",
        "tier": "COMMON"
      },
      {
        "id": "HYPERION",
        "name": "Hyperion",
        "material": "IRON_SWORD",
        "tier": "LEGENDARY"
      }
    ]
  },
  "meta": { "cached": true, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `items[].id` | string | Internal item ID |
| `items[].name` | string | Display name |
| `items[].material` | string | Minecraft material type |
| `items[].tier` | string | Rarity tier (COMMON, UNCOMMON, RARE, EPIC, LEGENDARY, MYTHIC, DIVINE, SPECIAL, VERY_SPECIAL) |
