## GET /v1/bazaar

Returns bazaar product data from the Hypixel API. Use the `mode` parameter to control the response format.

**Cache:** 5 seconds

### Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `mode` | string | `full` | Response mode: `full`, `list`, `item`, or `flips` |
| `id` | string | — | Bazaar item ID (**required** when `mode=item`) |
| `limit` | integer | `20` | Max results for `mode=flips` (1–100) |

---

### mode=list

Returns a sorted array of all active bazaar item IDs (items with non-zero prices).

```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar?mode=list"
```

```json
{
  "success": true,
  "data": {
    "count": 1200,
    "items": [
      { "id": "ABSOLUTE_ENDER_PEARL" },
      { "id": "ACACIA_FENCE" }
    ]
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

### mode=item

Returns processed data for a single item with spread calculations.

```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar?mode=item&id=DIAMOND"
```

```json
{
  "success": true,
  "data": {
    "id": "DIAMOND",
    "sellPrice": 7.5,
    "buyPrice": 8.1,
    "sellVolume": 1234567,
    "buyVolume": 987654,
    "spread": 0.6,
    "spreadPercent": 8.0,
    "sellMovingWeek": 50000000,
    "buyMovingWeek": 45000000
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `sellPrice` | number | Instant-sell price (what you get selling) |
| `buyPrice` | number | Instant-buy price (what you pay buying) |
| `spread` | number | `buyPrice - sellPrice` |
| `spreadPercent` | number | `(spread / sellPrice) * 100` |
| `sellVolume` / `buyVolume` | number | Current order volume |
| `sellMovingWeek` / `buyMovingWeek` | number | 7-day moving trade volume |

### mode=flips

Returns items sorted by `spreadPercent`, filtered for items with sell price >= 50,000 coins and volume >= 10,000.

```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar?mode=flips&limit=5"
```

```json

  "data": {
```

## GET /v1/bazaar/trends

Returns pre-computed trend signals for all bazaar items. Updated every 5 minutes.

**Cache:** 1 minute

```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar/trends"
```

```json
{
  "success": true,
  "data": {
    "trends": { ... },
    "updated_at": "2026-03-05T12:00:00Z"
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

<Aside>Trend data is computed on-server from 5-minute bazaar snapshots. It includes price movement signals across multiple timeframes.</Aside>

### Errors

| Code | HTTP Status | When |
|---|---|---|
| `NOT_FOUND` | 404 | `mode=item` with an invalid or non-existent `id` |
| `BAD_REQUEST` | 400 | Invalid `mode` value or missing `id` for item mode |
