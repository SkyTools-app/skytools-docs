## GET /v1/guild

Look up a Hypixel guild by its name or by a player's username. You must provide **one** of `name` or `player`.

**Cache:** 15 minutes

### Query Parameters

| Parameter | Type | Description |
|---|---|---|
| `name` | string | Guild name to search for |
| `player` | string | Minecraft username — returns their guild |

One of `name` or `player` is **required**.

### Examples

<Tabs>
  <TabItem label="cURL">
```bash
# Look up by guild name
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/guild?name=The+Bloodlust"

# Look up by player
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/guild?player=Refraction"
```
  </TabItem>
  <TabItem label="JavaScript">
```javascript
// By guild name
const response = await fetch(
  "https://api.skytools.app/v1/guild?name=The+Bloodlust",
  { headers: { "X-API-Key": "sk_your_key_here" } }
);
const { data } = await response.json();
console.log(`${data.guild.name} [${data.guild.tag}] — ${data.guild.members.length} members`);
```
  </TabItem>
  <TabItem label="Python">
```python
import requests

# By player username
response = requests.get(
    "https://api.skytools.app/v1/guild",
    params={"player": "Refraction"},
    headers={"X-API-Key": "sk_your_key_here"},
)
guild = response.json()["data"]["guild"]
print(f"{guild['name']} [{guild['tag']}]")
```

### Response

```json
{
  "success": true,
  "data": {
    "guild": {
      "name": "The Bloodlust",
      "tag": "BLOOD",
      "created": 1560000000000,
      "members": [
        {
          "uuid": "b876ec32e396476ba1158438d83c67d4",
          "rank": "Guild Master",
          "joined": 1560000000000
        }
      ],
      "exp": 50000000
    }
  },
  "meta": { "cached": false, "api_version": "v1" }
}
```

| Field | Type | Description |
|---|---|---|
| `guild.name` | string | Guild display name |
| `guild.tag` | string | Guild tag (shown in chat) |
| `guild.created` | number | Guild creation timestamp (Unix ms) |
| `guild.members` | array | Guild member list |
| `guild.members[].uuid` | string | Member's Minecraft UUID |
| `guild.members[].rank` | string | Member's guild rank |
| `guild.members[].joined` | number | Join timestamp (Unix ms) |
| `guild.exp` | number | Total guild experience |

### Errors

| Code | HTTP Status | When |
|---|---|---|
| `BAD_REQUEST` | 400 | Neither `name` nor `player` was provided |
| `NOT_FOUND` | 404 | No guild found with that name or the player isn't in a guild |
| `UPSTREAM_ERROR` | 502 | Hypixel API returned an error |
