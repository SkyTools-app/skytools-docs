The SkyTools API gives you programmatic access to real-time and historical Hypixel SkyBlock data — bazaar prices, auction listings, player profiles, net worth breakdowns, items, collections, mayors, fire sales, guilds, and crafting recipes.

**Base URL**

```
https://api.skytools.app/v1
```

## Quick Start

<Aside type="tip">All SkyTools accounts — including Free tier — can create API keys.</Aside>

1. Create a free account at [skytools.app](https://skytools.app)
2. Generate an API key in the [Developer Dashboard](/dashboard)
3. Start making requests:

<Tabs>
  <TabItem label="cURL">
```bash
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar?mode=list"
```
  </TabItem>
  <TabItem label="JavaScript">
```javascript
const response = await fetch("https://api.skytools.app/v1/bazaar?mode=list", {
  headers: { "X-API-Key": "sk_your_key_here" },
});
const { success, data, meta } = await response.json();
console.log(data.items);
```
  </TabItem>
  <TabItem label="Python">
```python
import requests

response = requests.get(
    "https://api.skytools.app/v1/bazaar",
    params={"mode": "list"},
    headers={"X-API-Key": "sk_your_key_here"},
)
data = response.json()
print(data["data"]["items"])
```
  </TabItem>
</Tabs>

## Response Format

Every response follows a consistent JSON envelope:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "cached": false,
    "api_version": "v1"
  }
}
```

Error responses use the same envelope:

```json
{
  "success": false,
  "error": {
    "code": "UNAUTHORIZED",
    "message": "API key is required. Get one at https://developers.skytools.app/dashboard"
  }
}
```

When the API serves stale cached data (e.g. during upstream timeouts), the `meta` object includes additional fields:

```json
{
  "meta": {
    "cached": true,
    "api_version": "v1",
    "stale": true,
    "age_seconds": 245
  }
}
```
