## API key required

All `/v1/*` endpoints require a valid API key. Requests without a key receive a `401 Unauthorized` response.

<Note>Every SkyTools account — including Free — can create API keys. Your tier only affects [rate limits](/rate-limits).</Note>

### Tiers

| | Free | Starter | Pro |
|---|---|---|---|
| API access | Yes | Yes | Yes |
| Rate limit | 60 req/min | 120 req/min | 300 req/min |
| Price | $0 | $5.99/mo | $9.99/mo |

## Getting an API key

1. Create a free account at [skytools.app](https://skytools.app)
2. Go to the [Developer Dashboard](https://developers.skytools.app/keys)
3. Sign in with your SkyTools account (email, Google, or Discord)
4. Click **Create API Key**
5. Copy your key — it starts with `sk_`

You can create multiple keys, rename them, and revoke them at any time from the dashboard.

## Sending your key

Include your API key using **either** of these methods:

<CodeGroup>
```bash X-API-Key Header (recommended)
curl -H "X-API-Key: sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar?mode=list"
```

```bash Bearer Token
curl -H "Authorization: Bearer sk_your_key_here" \
  "https://api.skytools.app/v1/bazaar?mode=list"
```
</CodeGroup>

### Code examples

<CodeGroup>
```javascript JavaScript
const API_KEY = "sk_your_key_here";

const response = await fetch("https://api.skytools.app/v1/bazaar?mode=list", {
  headers: { "X-API-Key": API_KEY },
});
const { success, data } = await response.json();
```

```python Python
import requests

API_KEY = "sk_your_key_here"

response = requests.get(
    "https://api.skytools.app/v1/bazaar",
    params={"mode": "list"},
    headers={"X-API-Key": API_KEY},
)
data = response.json()
```
</CodeGroup>

## Error responses

### No key provided

```json
{
  "success": false,
  "error": {
    "code": "UNAUTHORIZED",
    "message": "API key is required. Get one at https://developers.skytools.app/dashboard"
  }
}
```

### Invalid or revoked key

```json
{
  "success": false,
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Invalid or revoked API key."
  }
}
```

## Key security

- **Keep your key secret.** Never commit it to source control or expose it in client-side code.
- **Use environment variables** to store your key in applications.
- **Revoke compromised keys** immediately in the [Dashboard](https://developers.skytools.app/keys) and create a new one.
- **Subscription expiry:** If your subscription lapses, your keys remain valid but revert to Free tier rate limits (60 req/min). There is a 3-day grace period after expiration before the downgrade takes effect.
